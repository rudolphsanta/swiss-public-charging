This repository includes codes used to query, process, save, and visualize historical statuses of public charging infrastructure in Switzerland between December 2022 and December 2025. 
The real-time status information and source of these data can be found here: https://opendata.swiss/de/dataset/ladestationen-fuer-elektroautos. 
The dataset is saved in the following repository: https://doi.org/10.5281/zenodo.18324721
